# Software Development Application

This project is a C#/.NET application developed using object-oriented programming principles.

## Overview
The application focuses on structured software design, implementing core business logic, data validation, and user interaction in a maintainable and well-organised codebase.

## Key Features
- Object-oriented design using classes and methods
- Implementation of core business logic
- Input validation and structured program flow
- Clean, readable, and maintainable code

## Technologies Used
- C#
- .NET
- Visual Studio
- GitHub

This project demonstrates strong foundations in software development, OOP concepts, and structured application design.
